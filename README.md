# SUPERTRUNFO

*** SUPER TRUNFO - CIDADES ***

Um jogo de cartas em que duas cidades batalham para ver qual é a melhor!

COMO JOGAR:
1. Cadastre os dados de duas cidades (População, Área, Pontos Turísticos e PIB).
2. O jogo calculará outros atributos como PIB per capita e Densidade Populacional.
3. Escolha um dos atributos para a disputa.
4. A cidade com o valor mais alto (ou mais baixo, no caso da Densidade) no atributo escolhido vence a rodada!

ATRIBUTOS PARA DISPUTA:
- População
- Área
- Pontos Turísticos
- PIB
- PIB per capita
- Densidade Populacional
- SUPER PODER (uma combinação de todos os atributos)

Desafie um amigo e veja qual cidade é a mais poderosa!
